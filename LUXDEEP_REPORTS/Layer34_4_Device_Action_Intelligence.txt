1. OVERVIEW
--------------------------------------------------------------------------------
Layer 34.4 introduces Device Action Intelligence Preview — a read-only bridge
for planning device actions safely. It provides action planning, validation,
permission checking, and risk analysis without executing any actual device
actions.

Operation Mode: read_only_preview_only
Execution Rule: all device actions must be planned, not executed
Action Levels: not_ready → conditional → ready → action_ready
Risk Model: safe → warning → high_risk → critical

2. CAPABILITIES
--------------------------------------------------------------------------------
  - action_planning
  - action_validation
  - permission_validation
  - scheduled_action_planning
  - post_task_action_planning
  - risk_analysis

3. SUPPORTED ACTIONS
--------------------------------------------------------------------------------
  notification, scheduled_action, post_task_action, system_reminder
  (shutdown, sleep, restart — available for preview only, never executed)

4. NOT ALLOWED OPERATIONS
--------------------------------------------------------------------------------
  shutdown | sleep | restart | execute_action | modify_system | delete_data
