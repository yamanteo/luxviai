1. OVERVIEW
--------------------------------------------------------------------------------
Layer 33.5 introduces Sandbox Repair Intelligence Preview — a read-only
analysis system for the LuxCode Safe Repair Execution Model.

Sandbox Execution Model:
  REAL_SYSTEM → MASTER_CLONE → WORKING_CLONE → SANDBOX → VALIDATION
  → DELIVERY_CANDIDATE

Repair Rules:
  - No repair in REAL_SYSTEM or MASTER_CLONE
  - All repairs in WORKING_CLONE only
  - All testing in SANDBOX only

Cancellation Model:
  - Discard Working Clone + Sandbox state
  - Regenerate from Master Clone
  - No rollback required. No production impact.

Loop Protection:
  - Detects repeated repair loops using Layers 33.1, 33.2, 33.3 data
  - Flags repeated failed repairs, temporary fixes, symptom-only fixes
