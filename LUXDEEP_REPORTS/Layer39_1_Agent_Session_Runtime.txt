1. OVERVIEW

---

Layer 39.1 introduces Agent Session Runtime.

This layer becomes the session lifecycle engine of LuxCode.

Primary Goal:

Manage the complete lifecycle of an active working session.

Primary Mission:

Know:

* when a session starts
* when a session pauses
* when a session resumes
* when a session changes context
* when a session ends

without losing execution continuity.
