1. OVERVIEW

---

Layer 39.5 introduces Agent Lifecycle Runtime.

This layer becomes the lifecycle management engine of LuxCode.

Primary Goal:

Manage the complete life journey of an active agent.

Primary Mission:

Understand:

* when an agent starts
* when an agent becomes active
* when an agent enters waiting mode
* when an agent pauses
* when an agent resumes
* when an agent shuts down
* when an agent recovers

while preserving operational continuity.
