1. OVERVIEW
--------------------------------------------------------------------------------
Layer 33.4 introduces Clone Workspace Intelligence Preview — a read-only
analysis system for the LuxCode Safe Execution Architecture workspace model.

Workspace Architecture:
  REAL_SYSTEM → MASTER_CLONE → WORKING_CLONE → SANDBOX

Master Clone Rules:
  - Reference-only; no repair, modification, experimentation, testing, patching
  - Continuously synchronized with real system (files, config, code, deps, docs)

Working Clone Rules:
  - All changes occur only in Working Clone
  - May be created, replaced, discarded, or regenerated at any time

Cancellation Model:
  - User/system cancellation supported
  - Result: discard Working Clone → rebuild from Master Clone → clean state
  - No rollback required. No production impact.
